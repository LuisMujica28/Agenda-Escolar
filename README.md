# Agenda Escolar Lite

Este proyecto es una aplicación web (PWA) para la gestión de comunicaciones escolares.

## 📁 Ubicación del Proyecto
El código fuente se encuentra guardado localmente en tu máquina en:
`C:\Users\USER\.gemini\antigravity\scratch\agenda_virtual_3`

## 🚀 Cómo correr el proyecto
Si cierras la terminal, puedes volver a iniciar la aplicación así:

1. Abrir una terminal (PowerShell o CMD).
2. Navegar a la carpeta:
   ```bash
   cd C:\Users\USER\.gemini\antigravity\scratch\agenda_virtual_3
   ```
3. Ejecutar el servidor:
   ```bash
   npm run dev -- --host
   ```
4. Abrir en el navegador: `http://localhost:5173`
